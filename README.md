## Features
- Dynamic product rendering from JS data array
- Real-time search + category + price + sort filters
- Countdown timer to drop close
- Persistent cart via localStorage (survives refresh)
- Live stock deduction as items are added
- Product detail modal with size selection
- Wishlist with localStorage persistence
- Discount codes: DROP10 (10%), CX20 (20%)
- Bootstrap form validation on checkout
- Fully responsive — 2-col mobile, 4-col desktop
